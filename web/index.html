Project Description

The project dealt with the creation of a website, where movies can be reviewed by the viewers. Website had “Sign-in” and “Create Account” pages where users could create an account or sign into their account. Once signed in, viewers could add their reviews, rate, add a movie to the website, update/delete reviews and delete a movie based on assigned security (roles). Users who are assigned an admin can add/remove movies and can edit or remove user reviews. A database was created to store/retrieve movie details, reviews, user account information, roles, etc. The website processed information to and from this database based on website activities.
